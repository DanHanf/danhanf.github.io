chs-tonight

Github

chs-tonight is a tool that scrapes the events listings and calendars of several bars, theaters, and venue spots in Charleston, SC. The scrapers run every night, then that data is used to build a new HTML page with details and links to the events. I have a series of blog posts that detail the development.



CitiBike Visualization

Github

Citibike hosts open ride data, which I used to make a few trip visualizations in NYC. This project was inspired by nyc-taxi and, while now outdated, was still fun to make and pretty to look at. Here's another visualization for Manhattan and Brooklyn.
